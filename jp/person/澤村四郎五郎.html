澤村四郎五郎 (Shirogoro SAWAMURA)

澤村 四郎五郎（さわむら しろうごろう）は、歌舞伎役者の名跡。

歌舞伎で「四郎五郎」の名は他に中村屋 (歌舞伎)系の中村四郎五郎がある。

澤村四郎五郎 (初代)
- 後の助高屋高助 (2代目)。

澤村四郎五郎 (2代目)
-澤村宗十郎(３代）の女婿。
屋号は川滝屋。
俳名は春風、遮莫。

荻野東蔵 → 澤村東蔵 → 2代目澤村四郎五郎 → 澤村東十郎 → 2代目澤村四郎五郎 → 澤村しやばく → 2代目澤村四郎五郎。

澤村四郎五郎 (3代目)
- 2代目市川友蔵（市川源左衛門）の子。
屋号は三河屋、川滝屋。
俳名は市升、遮莫。

市川團子（代々の團子とは代外） → 市川子之助 → 3代目市川友蔵 → 市川團九郎 → 市川一友 → 3代目澤村四郎五郎。

澤村四郎五郎 (4代目)
- 助高屋高助 (4代目)の養子か？
嵐三之助の婿養子、兄に中村鶴三郎、中村飛鶴がいる。
本名は代田多吉。
屋号は紀伊國屋 (歌舞伎)。

實川雁五郎 → 4代目澤村四郎五郎。

澤村四郎五郎 (5代目)
- 後に映画俳優に転身。
屋号は紀伊国屋。
俳名は三鵆。

中村福次 → 澤村國之助 → 5代目澤村四郎五郎。
本名は沢野乙吉。
